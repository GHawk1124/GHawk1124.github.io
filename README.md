# Garrett Comes Portfolio

Astro portfolio site for [garrettcomes.me](https://garrettcomes.me).

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The build emits static files to `dist/`. GitHub Pages serves the built artifact.

## Preserved Demos

The static demos in `public/fem_rust_ae/` and `public/simulator/` are copied into
the build output unchanged so their public paths remain available:

- `/fem_rust_ae/`
- `/simulator/sim.html`
